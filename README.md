### Local development

Install [node 8](https://nodejs.org/dist/latest-v8.x/) and use the following

```shell
  npm install
    Install dependencies

  npm run dev
    Starts the development server.

  npm run build
    Bundles the app into static files for production.

  npm run test
    Run tests for project
```
