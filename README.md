### Local development

Install [yarn](https://yarnpkg.com/en/docs/install) and use the following

```shell
  yarn
    Install dependencies

  yarn dev
    Starts the development server.

  yarn build
    Bundles the app into static files for production.

  yarn test
    Run tests for project
```
